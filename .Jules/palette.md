## 2026-04-10 - Added HTML lang and noscript fallback for Flutter Web\n**Learning:** Flutter Web applications mount directly into an almost empty index.html, so basic accessibility features like screen reader language targeting require explicit HTML attributes (e.g., `<html lang="tr">`), and fallback content requires explicit `<noscript>` tags because Flutter relies completely on JavaScript.\n**Action:** Always check the base `index.html` of Flutter Web projects to ensure the document language is correctly set for screen readers and a graceful fallback is provided for environments where JS is disabled.
